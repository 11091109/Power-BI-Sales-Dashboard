* Sales Performance Dashboard



1. **Project Overview:**



This Power BI dashboard is designed to analyze business sales performance and track key business indicators through interactive visualizations.



**2.  Business Objective:**



The objective of this project is to monitor overall sales performance, identify profitable regions, and evaluate conversion efficiency for better business decision-making.



**3.  Key Performance Indicators (KPIs):**



**\*** Total Revenue

**\*** Total Profit

**\*** Conversion Rate

**\*** Total Orders



**4. Dashboard Features:**



**\*** KPI cards for quick performance tracking

**\*** Sales comparison by category

**\*** Regional sales performance analysis

**\*** Profit contribution visualization

**\*** Conversion rate monitoring



**5. Tools Used:**



**\*** Microsoft Power BI

**\*** Microsoft Excel



**6. Dataset Used:**



The dashboard is built using structured sales data stored in Excel format.
